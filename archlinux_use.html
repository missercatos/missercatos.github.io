开机后第一件事
安装完成后重启，登录到你创建的用户，首先确保网络已连接，然后更新系统到最新状态。
连接网络（如果 NetworkManager 已安装）：
nmtui — 打开终端图形化网络管理工具
或使用命令：nmcli device wifi connect "WiFi名" password "密码"

更新系统：
sudo pacman -Syu

这是 Arch Linux 最重要的命令之一，可以每次开机后先执行一次，保持系统处于最新状态（滚动更新）。
但是要了解突然更新会导致环境崩坏的可能，因此要做好快照备份，这个我在后期会有介绍和教学
桌面环境推荐
如果你在安装时没有选择桌面环境，或者想更换桌面环境，以下是各桌面环境的对比和安装方法。
KDE Plasma — 新手首选
功能最全面的桌面环境，界面美观现代，高度可定制，操作习惯接近 Windows。
安装命令：
sudo pacman -S plasma kde-applications sddm
sudo systemctl enable sddm（设置开机自动启动）
reboot（重启后即可看到 KDE 登录界面）
GNOME — 简洁优雅
简洁流畅的设计风格，操作逻辑类似 macOS，触屏和手势支持极好。
安装命令：
sudo pacman -S gnome gnome-extra gdm
sudo systemctl enable gdm
reboot
Xfce — 轻量高效
资源占用极低，适合老旧电脑或追求极致性能的用户，功能也足够日常使用。
安装命令：
sudo pacman -S xfce4 xfce4-goodies lightdm lightdm-gtk-greeter
sudo systemctl enable lightdm
Hyprland — 酷炫平铺窗口
动态平铺式窗口管理器，动画丝滑，适合追求个性和效率的高级用户。
安装命令：
sudo pacman -S hyprland kitty waybar wofi

常用软件
Arch Linux 安装完成后是"纯净"的，你需要自己安装日常使用的软件。
使用 sudo pacman -S 包名 即可安装，以下是分类推荐：
🌐 网络浏览器：
  • sudo pacman -S firefox — Firefox 浏览器（推荐，开源）
  • sudo pacman -S chromium — Chromium 浏览器（开源版 Chrome）
  • 安装 Chrome：通过 AUR：yay -S google-chrome
📝 文本编辑器：
  • sudo pacman -S vim — Vim（终端编辑器之神）
  • sudo pacman -S neovim — Neovim（现代化的 Vim）
  • sudo pacman -S code — VS Code（最流行的图形化编辑器）
  • sudo pacman -S kate — Kate（KDE 自带，轻量好用）
🎬 影音播放：
  • sudo pacman -S vlc — VLC（万能播放器）
  • sudo pacman -S mpv — MPV（轻量高性能播放器）
  • sudo pacman -S strawberry — Strawberry（音乐播放器）
💬 通讯社交：
  • sudo pacman -S discord — Discord
  • sudo pacman -S telegram-desktop — Telegram
  • QQ/微信：可通过 AUR 安装：yay -S icu69-bin 后 yay -S deepin-wine-qq
🎮 游戏相关：
  • sudo pacman -S steam — Steam（开启 Proton 即可玩 Windows 游戏）
  • sudo pacman -S lutris — Lutris（游戏管理平台）
  • sudo pacman -S wine — Wine（运行 Windows 程序）
🛠 开发工具：
  • sudo pacman -S base-devel — 编译工具链（gcc, make 等）
  • sudo pacman -S git — Git 版本控制
  • sudo pacman -S python — Python
  • sudo pacman -S nodejs npm — Node.js + npm
  • sudo pacman -S docker — Docker 容器

AUR 助手
AUR（Arch User Repository）是 Arch Linux 最强大的特性之一。官方仓库没有的软件，AUR 几乎都有。
yay 是最流行的 AUR 助手，使用方法与 pacman 几乎一样。
当然，aur助手也是由人开发出来的。大致原理是获取aur仓库的引导文件，下载并安装,自动配置环境，理论上人人都能自己编写一个aur助手，这里直接用现成的就好
安装 yay：
sudo pacman -S --needed git base-devel
git clone https://aur.archlinux.org/yay.git
cd yay
makepkg -si
yay 常用命令：
yay -S 包名 — 安装软件（自动搜索官方库和 AUR）
yay -Syu — 更新整个系统（包括 AUR 包）
yay -Ss 关键词 — 搜索软件包
yay -Rns 包名 — 彻底卸载软件及无用依赖
yay -Yc — 清理无用的 AUR 构建缓存

中文输入法
在 Arch Linux 上使用中文输入法需要安装输入法框架和输入法引擎。
推荐方案：Fcitx5（最成熟的 Linux 中文输入方案）

安装：
sudo pacman -S fcitx5 fcitx5-chinese-addons fcitx5-qt fcitx5-gtk fcitx5-configtool

配置环境变量（在 ~/.bashrc 或 ~/.xprofile 中添加）：
export GTK_IM_MODULE=fcitx
export QT_IM_MODULE=fcitx
export XMODIFIERS=@im=fcitx
export SDL_IM_MODULE=fcitx
重启或重新登录后，使用 Ctrl+Space 切换中英文输入。

日常维护
Arch Linux 是滚动更新发行版，定期维护可以保证系统稳定运行。
📦 Pacman 常用命令速查：
  • sudo pacman -Syu — 全面更新系统（最常用）
  • sudo pacman -S 包名 — 安装软件包
  • sudo pacman -R 包名 — 卸载软件包（保留依赖）
  • sudo pacman -Rns 包名 — 彻底卸载（删除依赖和配置）
  • sudo pacman -Ss 关键词 — 搜索软件包
  • sudo pacman -Q — 列出所有已安装的包
  • sudo pacman -Qdt — 列出不再需要的孤立包
  • sudo pacman -Sc — 清理未安装包的缓存
  • sudo pacman -Scc — 清理全部缓存（谨慎使用）

🧹 定期清理建议：
sudo pacman -Rns $(pacman -Qdtq) — 移除孤立包
sudo journalctl --vacuum-size=100M — 限制日志大小
paccache -r — 保留最近3个版本的包缓存（需先装 pacman-contrib）
更新前建议查看 Arch Linux 官方新闻，了解是否有需要手动干预的更新。
小技巧
一些能让你使用 Arch Linux 更顺手的小技巧和工具。
🔧 终端神器推荐：
  • sudo pacman -S fastfetch — 系统信息显示（替代 neofetch）
  • sudo pacman -S btop — 漂亮的系统资源监控（类任务管理器）
  • sudo pacman -S ranger — 终端文件管理器
  • sudo pacman -S tldr — 命令速查手册（比 man 更简洁）
  • sudo pacman -S tmux — 终端多窗口复用
  • sudo pacman -S zsh — 比 bash 更好用的 shell

🎨 配置 Zsh：
sudo pacman -S zsh zsh-completions
chsh -s /bin/zsh（切换默认 shell）
然后安装 oh-my-zsh 美化终端：
sh -c "$(curl -fsSL https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
