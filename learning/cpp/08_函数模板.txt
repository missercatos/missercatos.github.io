一、模板基础语法

模板使用 template 关键字声明，后面跟模板参数列表。类型参数可以使用
typename 或 class 关键字（两者在模板参数声明中完全等价）。

---------------------------------------------------------------
// ---- 基础示例：通用最大值函数 ----
#include <iostream>
#include <string>

// 函数模板定义
template<typename T>
T maxValue(T a, T b) {
    return (a > b) ? a : b;
}

int main() {
    // 隐式实例化：编译器根据实参类型自动推导 T
    std::cout << maxValue(3, 7) << std::endl;          // T = int
    std::cout << maxValue(3.14, 2.71) << std::endl;    // T = double
    std::cout << maxValue(std::string("hello"),
                          std::string("world")) << std::endl;  // T = std::string

    // 显式实例化
    std::cout << maxValue<int>(5.5, 3.3) << std::endl;  // T = int, 浮点数被截断

    return 0;
}
---------------------------------------------------------------

二、编译器底层原理详解

1. 编译期多态 vs 运行时多态

   ┌──────────────────────────────────────────────────────────────┐
   │              编译期多态 (模板)      vs      运行时多态 (虚函数) │
   ├──────────────────────────────────────────────────────────────┤
   │ 绑定时机:   编译期                      运行时                  │
   │ 实现机制:   代码生成(实例化)           虚函数表(vtable)         │
   │ 性能开销:   无运行时开销,但代码膨胀     有间接调用开销           │
   │ 灵活性:     必须在编译期确定类型        可在运行期扩展类型       │
   │ 典型场景:   STL容器/算法               插件系统/框架设计         │
   └──────────────────────────────────────────────────────────────┘

   模板被称作"静态多态"或"编译期多态"：多态行为在编译阶段已完全确定，
   不依赖虚函数表（vtable），因此没有虚函数调用的间接寻址开销。

2. 模板实例化机制深度解析

   关键原则：编译器看到模板定义时**不生成任何代码**。
   只有当模板被调用（实例化）时，编译器才根据实际的模板实参生成具体的函数。

   实例化过程分步解析：

   步骤1: 编译器解析模板定义，进行基本语法检查（第一阶段检查）。
          例如检查括号匹配、关键字正确性等与模板参数无关的语法。
          这个阶段也叫"定义时检查"。

   步骤2: 当编译器遇到模板调用时，例如 maxValue(3, 7)，
          编译器开始第二阶段检查——"实例化时检查"。
          编译器生成 int maxValue(int, int) 的完整函数定义。

   步骤3: 编译器将实例化后的代码编译为目标代码（.o文件）。

   这个"两阶段编译检查"模型（Two-Phase Lookup）是C++模板的核心机制：
   - 第一阶段：检查不依赖模板参数的代码（非依赖名）
   - 第二阶段：实例化时检查依赖模板参数的代码（依赖名）

   示例说明：

   template<typename T>
   void foo(T x) {
       x.hello();        // 第一阶段: 不做检查（x的类型未知，属于依赖名）
       int y = "abc";    // 第一阶段: 报错！（不依赖T，应在此阶段检查）
   }

3. 代码膨胀 (Code Bloat)

   模板的一个潜在问题是代码膨胀：

   template<typename T>
   void process(T value) { /* 100行代码 */ }

   process<int>(1);      // 生成一份100行的int版本
   process<double>(1.0); // 又生成一份100行的double版本
   process<float>(1.0f); // 又生成一份100行的float版本

   每种不同的模板参数组合都会产生一份独立的代码副本。
   如果模板函数体积大且实例化类型多，最终二进制体积会显著增大。

   缓解策略：
   - 将不依赖模板参数的代码提取到非模板基类中（瘦模板技术）
   - 使用 void* 实现类型擦除，只实例化一次
   - 在C++20中使用 concepts 约束实例化的类型范围

4. 模板在目标文件中的处理

   同一个模板函数在多个编译单元（.cpp文件）中被实例化时，
   每个编译单元都会生成一份该实例化的目标代码。
   例如 a.cpp 和 b.cpp 都调用了 maxValue<int>(1,2)，
   则 a.o 和 b.o 中各有一份 maxValue<int> 的机器码。

   链接器在最终链接时负责"去重"（也称为 COMDAT 折叠或 ICF），
   只保留一份代码。这意味着：
   - 编译时间增加（每个.cpp都要实例化一次）
   - 但最终二进制文件不会重复膨胀

5. 汇编层面的模板

   实例化后的模板函数与普通函数在汇编层面没有区别。
   以下面的代码为例：

   template<typename T>
   T square(T x) { return x * x; }

   square(5);    // 实例化为 int square(int)
   square(3.14); // 实例化为 double square(double)

    编译器生成的汇编代码中，square(5) 和 square(3.14) 是
    两个完全独立的函数，有不同的函数名修饰（name mangling），
    其地址也完全不同。模板在汇编层面没有特殊标记。

6. 模板的编译期分支 —— if constexpr (C++17)

   C++17 引入的 if constexpr 允许在编译期进行条件分支。
   与普通 if 不同，if constexpr 的条件在编译期求值，
   不满足条件的分支不会被实例化（甚至不需要语法正确）。

   这解决了模板代码中"同一份代码适配不同类型"的难题：

   ---------------------------------------------------------------
   #include <iostream>
   #include <type_traits>
   #include <string>

   template<typename T>
   auto getValueDescription(const T& value) {
       if constexpr (std::is_integral_v<T>) {
           return "整数: " + std::to_string(value);
       }
       else if constexpr (std::is_floating_point_v<T>) {
           return "浮点数: " + std::to_string(value);
       }
       else if constexpr (std::is_same_v<T, std::string>) {
           return "字符串: " + value;
       }
       else {
           return "未知类型 (大小: " + std::to_string(sizeof(T)) + " 字节)";
       }
   }

   int main() {
       std::cout << getValueDescription(42) << std::endl;
       std::cout << getValueDescription(3.14) << std::endl;
       std::cout << getValueDescription(std::string("Hello")) << std::endl;
       std::cout << getValueDescription(nullptr) << std::endl;
       return 0;
   }
   ---------------------------------------------------------------

   if constexpr 的关键特性：
   - 条件必须是编译期常量表达式
   - 未选择的分支完全不会被编译（不会引发语法错误）
   - 常用于消除 SFINAE 的笨重语法

7. 显式实例化 (Explicit Instantiation)

   当模板定义在.cpp文件中，为了控制实例化位置或减少编译时间，
   可以使用显式实例化：

   ---------------------------------------------------------------
   // ---- mytemplate.h ----
   template<typename T>
   T max(T a, T b) {
       return (a > b) ? a : b;
   }

   extern template int max<int>(int, int);      // 声明：不要在此编译单元实例化
   extern template double max<double>(double, double);

   // ---- mytemplate.cpp ----
   template int max<int>(int, int);             // 显式实例化定义
   template double max<double>(double, double);  // 在此处生成代码
   ---------------------------------------------------------------

   使用场景：当多个.cpp文件使用相同的模板实例化时，
   用 extern template 声明抑制自动实例化，在一个.cpp中
   集中显式实例化，可以减少总编译时间。

8. 模板参数推导指南 (Deduction Guides, C++17)

   类模板的模板参数在 C++17 起可以自动推导：

   ---------------------------------------------------------------
   #include <vector>
   #include <utility>

   int main() {
       std::vector v = {1, 2, 3, 4, 5};   // 推导为 vector<int> (C++17)
       std::pair p(3.14, "pi");            // 推导为 pair<double, const char*>

       // 自定义推导指南：
       // template<typename T>
       // MyContainer(std::initializer_list<T>) -> MyContainer<T>;

       return 0;
   }
   ---------------------------------------------------------------

9. 模板与编译期类型计算 —— type_traits 深探

   <type_traits> 提供了大量编译期类型查询和转换工具：

   ---------------------------------------------------------------
   #include <iostream>
   #include <type_traits>
   #include <vector>

   template<typename T>
   void analyzeType() {
       std::cout << "类型 " << typeid(T).name() << " 的属性:" << std::endl;
       std::cout << "  是整数:    " << std::is_integral_v<T> << std::endl;
       std::cout << "  是浮点:    " << std::is_floating_point_v<T> << std::endl;
       std::cout << "  是类:      " << std::is_class_v<T> << std::endl;
       std::cout << "  是指针:    " << std::is_pointer_v<T> << std::endl;
       std::cout << "  是const:   " << std::is_const_v<T> << std::endl;
       std::cout << "  大小:      " << sizeof(T) << " 字节" << std::endl;

       // 类型转换示例
       using RemovedRef = std::remove_reference_t<T>;
       using AddedConst = std::add_const_t<T>;
       using Decayed = std::decay_t<T>;  // 移除引用/const/volatile，数组转指针

       std::cout << "  decay_t   = " << typeid(Decayed).name() << std::endl;
       std::cout << std::endl;
   }

   int main() {
       analyzeType<int>();
       analyzeType<double>();
       analyzeType<const int&>();
       analyzeType<std::vector<int>>();
       analyzeType<int[10]>();
       return 0;
   }
   ---------------------------------------------------------------

   常用 type_traits 分类：
   ┌─────────────────────┬──────────────────────────────────────────┐
   │ 分类                │ 示例 trait                                │
   ├─────────────────────┼──────────────────────────────────────────┤
   │ 类型查询            │ is_void, is_integral, is_floating_point  │
   │ 类型关系            │ is_same, is_base_of, is_convertible      │
   │ 类型修改            │ remove_const, add_pointer, decay         │
   │ 类型属性            │ is_pod, is_abstract, is_trivially_xxx    │
   │ 复合类型            │ is_reference, is_pointer, is_array       │
   │ SFINAE支持          │ enable_if, conditional, void_t           │
   └─────────────────────┴──────────────────────────────────────────┘

一、函数模板定义

基本形式：

template<typename T>       // 推荐使用 typename（语义更清晰）
T add(T a, T b) { return a + b; }

template<class T>           // class 也可以，与 typename 等价
T subtract(T a, T b) { return a - b; }

// 多个模板参数
template<typename T1, typename T2>
auto multiply(T1 a, T2 b) -> decltype(a * b) {
    return a * b;
}

// C++14 起可以用 auto 自动推导返回类型
template<typename T1, typename T2>
auto multiply14(T1 a, T2 b) {
    return a * b;    // 编译器自动推导返回类型
}

二、非类型模板参数 (Non-type Template Parameters)

除了类型参数，模板还可以接受编译期常量值作为参数：

---------------------------------------------------------------
#include <iostream>
#include <array>

// N 是一个编译期常量
template<typename T, int N>
class FixedArray {
private:
    T data[N];          // 数组大小在编译期确定，无需动态内存
public:
    int size() const { return N; }
    T& operator[](int index) { return data[index]; }
    const T& operator[](int index) const { return data[index]; }
    void fill(const T& value) {
        for(int i = 0; i < N; ++i) data[i] = value;
    }
};

int main() {
    FixedArray<int, 5> arr;     // 5在编译期确定
    arr.fill(42);
    std::cout << arr[3] << std::endl;  // 输出 42
    return 0;
}
---------------------------------------------------------------

非类型模板参数可以是：
- 整型常量（int, long, short, char, bool 等）
- 枚举常量
- 指针/引用（指向全局对象或函数）
- std::nullptr_t
- C++20 起：浮点数、类类型（字面量类型）

三、类模板定义

类模板是泛型容器的基础（如 std::vector<T>, std::map<K,V>）：

---------------------------------------------------------------
#include <iostream>
#include <stdexcept>

template<typename T>
class Stack {
private:
    T* data;
    int capacity;
    int topIndex;
public:
    explicit Stack(int cap = 100)
        : capacity(cap), topIndex(-1) {
        data = new T[capacity];      // 动态内存分配
    }

    ~Stack() { delete[] data; }

    Stack(const Stack& other)        // 拷贝构造
        : capacity(other.capacity), topIndex(other.topIndex) {
        data = new T[capacity];
        for(int i = 0; i <= topIndex; ++i)
            data[i] = other.data[i];
    }

    Stack& operator=(const Stack& other) {
        if(this != &other) {
            delete[] data;
            capacity = other.capacity;
            topIndex = other.topIndex;
            data = new T[capacity];
            for(int i = 0; i <= topIndex; ++i)
                data[i] = other.data[i];
        }
        return *this;
    }

    void push(const T& value) {
        if(topIndex >= capacity - 1)
            throw std::overflow_error("Stack overflow");
        data[++topIndex] = value;
    }

    T pop() {
        if(topIndex < 0)
            throw std::underflow_error("Stack underflow");
        return data[topIndex--];
    }

    T& top() {
        if(topIndex < 0)
            throw std::underflow_error("Stack empty");
        return data[topIndex];
    }

    bool empty() const { return topIndex < 0; }
    int size() const { return topIndex + 1; }
};

int main() {
    Stack<std::string> s(10);
    s.push("Hello");
    s.push("World");
    std::cout << s.pop() << std::endl;  // World
    std::cout << s.pop() << std::endl;  // Hello
    return 0;
}

四、模板特化 (Template Specialization)

4.1 全特化 (Full Specialization)

当某种特定类型需要特殊处理时，可以对模板进行完全特化：

---------------------------------------------------------------
#include <iostream>

// 通用模板
template<typename T>
class TypeInfo {
public:
    static const char* name() { return "Unknown"; }
    static int size() { return sizeof(T); }
};

// 对 int 的全特化
template<>
class TypeInfo<int> {
public:
    static const char* name() { return "integer"; }
    static int size() { return sizeof(int); }
};

// 对 double 的全特化
template<>
class TypeInfo<double> {
public:
    static const char* name() { return "double-precision float"; }
    static int size() { return sizeof(double); }
};

// 对 std::string 的全特化
template<>
class TypeInfo<std::string> {
public:
    static const char* name() { return "string"; }
    static int size() { return sizeof(std::string); }
};

int main() {
    std::cout << TypeInfo<char>::name() << std::endl;      // Unknown
    std::cout << TypeInfo<int>::name() << std::endl;       // integer
    std::cout << TypeInfo<double>::name() << std::endl;    // double-precision float
    return 0;
}
---------------------------------------------------------------

4.2 偏特化 (Partial Specialization)

偏特化允许对模板参数的一部分进行特化，或对参数添加约束：

---------------------------------------------------------------
#include <iostream>

// 通用模板
template<typename T, typename U>
class Pair {
public:
    T first;
    U second;
    Pair(T f, U s) : first(f), second(s) {}
    void describe() { std::cout << "Generic Pair" << std::endl; }
};

// 偏特化：两个类型相同时
template<typename T>
class Pair<T, T> {
public:
    T first, second;
    Pair(T f, T s) : first(f), second(s) {}
    void describe() { std::cout << "Homogeneous Pair" << std::endl; }
};

// 偏特化：第二个参数固定为 int
template<typename T>
class Pair<T, int> {
public:
    T first;
    int second;
    Pair(T f, int s) : first(f), second(s) {}
    void describe() { std::cout << "Pair with int second" << std::endl; }
};

// 偏特化：指针类型（去掉指针，保留指向类型）
template<typename T>
class Pair<T*, T*> {
public:
    T* first;
    T* second;
    Pair(T* f, T* s) : first(f), second(s) {}
    void describe() { std::cout << "Pointer Pair" << std::endl; }
};

int main() {
    Pair<double, char> p1(3.14, 'A');    // 通用版本
    p1.describe();                         // "Generic Pair"

    Pair<int, int> p2(1, 2);              // 偏特化: 同类型版本
    p2.describe();                         // "Homogeneous Pair"

    Pair<double, int> p3(3.14, 42);       // 偏特化: 第二个是 int
    p3.describe();                         // "Pair with int second"

    int a = 10, b = 20;
    Pair<int*, int*> p4(&a, &b);          // 偏特化: 指针版本
    p4.describe();                         // "Pointer Pair"

    return 0;
}

五、默认模板参数

如同函数可以有默认参数，模板也可以有默认模板参数：

---------------------------------------------------------------
#include <iostream>
#include <vector>
#include <functional>

// 默认模板参数：Container 默认为 std::vector
template<typename T, typename Container = std::vector<T>>
class Queue {
private:
    Container data;
public:
    void enqueue(const T& value) { data.push_back(value); }
    T dequeue() {
        T front = data.front();
        data.erase(data.begin());
        return front;
    }
    bool empty() const { return data.empty(); }
    size_t size() const { return data.size(); }
};

// 比较器默认参数
template<typename T, typename Comparator = std::less<T>>
T maxOfThree(T a, T b, T c, Comparator cmp = Comparator()) {
    T maxVal = a;
    if(cmp(maxVal, b)) maxVal = b;
    if(cmp(maxVal, c)) maxVal = c;
    return maxVal;
}

int main() {
    // 使用默认容器 std::vector
    Queue<int> q;
    q.enqueue(1);
    q.enqueue(2);
    std::cout << q.dequeue() << std::endl;  // 1

    // 使用默认比较器 std::less
    std::cout << maxOfThree(3, 7, 1) << std::endl;  // 7

    return 0;
}

六、可变参数模板 (Variadic Templates, C++11)

可变参数模板让函数或类可以接受任意数量、任意类型的参数：

---------------------------------------------------------------
#include <iostream>
#include <string>

// 递归终止条件：空参数包
void print() {
    std::cout << std::endl;  // 最后一个参数后换行
}

// 递归展开参数包
template<typename T, typename... Args>
void print(T first, Args... rest) {
    std::cout << first;
    if constexpr(sizeof...(rest) > 0)   // C++17 if constexpr
        std::cout << ", ";
    print(rest...);  // 递归调用，参数包减少一个
}

// 使用初始化列表展开（另一种展开方式）
template<typename... Args>
void printAll(Args... args) {
    // 使用逗号表达式和初始化列表展开
    int dummy[] = { (std::cout << args << " ", 0)... };
    std::cout << std::endl;
    (void)dummy;  // 消除未使用变量警告
}

// 另一个递归示例：求和
template<typename T>
T sum(T value) {
    return value;
}

template<typename T, typename... Args>
T sum(T first, Args... rest) {
    return first + sum(rest...);
}

int main() {
    print("Hello", 42, 3.14, std::string("World"));
    // 输出: Hello, 42, 3.14, World

    printAll(1, 2, 3, 4, 5);
    // 输出: 1 2 3 4 5

    std::cout << sum(1, 2, 3, 4, 5) << std::endl;         // 15
    std::cout << sum(1.5, 2.5, 3.0) << std::endl;         // 7.0

    return 0;
}
---------------------------------------------------------------

七、折叠表达式 (Fold Expressions, C++17)

折叠表达式是处理可变参数模板的简洁语法，分为四种形式：

---------------------------------------------------------------
#include <iostream>
#include <string>

// 一元左折叠：(... op args)
template<typename... Args>
auto sumLeft(Args... args) {
    return (... + args);  // 展开为 ((arg1 + arg2) + arg3) + ...
}

// 一元右折叠：(args op ...)
template<typename... Args>
auto sumRight(Args... args) {
    return (args + ...);  // 展开为 arg1 + (arg2 + (arg3 + ...))
}

// 二元左折叠：(init op ... op args)
template<typename... Args>
auto sumWithInit(Args... args) {
    return (0 + ... + args);  // 初始值为0
}

// 二元右折叠：(args op ... op init)
template<typename... Args>
auto prodWithInit(Args... args) {
    return (args * ... * 1);  // 初始值为1
}

// 逗号运算符折叠：依次执行每个表达式
template<typename... Args>
void printAllFold(Args... args) {
    (std::cout << ... << args) << std::endl;  // 二元左折叠
}

// 逻辑与折叠：所有条件都满足
template<typename... Args>
bool allTrue(Args... args) {
    return (... && args);  // 所有参数都为真时返回真
}

// 逻辑或折叠：任一条件满足
template<typename... Args>
bool anyTrue(Args... args) {
    return (... || args);
}

int main() {
    std::cout << "Sum: " << sumLeft(1, 2, 3, 4, 5) << std::endl;      // 15
    std::cout << "Sum right: " << sumRight(1, 2, 3) << std::endl;     // 6
    std::cout << "Sum with init: " << sumWithInit(10, 20) << std::endl;// 30
    std::cout << "Product: " << prodWithInit(2, 3, 4) << std::endl;   // 24

    printAllFold("Hello", " ", "World", " ", 2024);
    // 输出: Hello World 2024

    std::cout << "All true: " << allTrue(true, 1, 5 > 3) << std::endl;      // 1 (true)
    std::cout << "Any true: " << anyTrue(false, 0, true) << std::endl;      // 1 (true)

    return 0;
}

八、模板元编程简介 (Template Metaprogramming)

模板元编程（TMP）利用模板在编译期执行计算，以阶乘为例：

---------------------------------------------------------------
#include <iostream>

// 模板元编程：编译期计算阶乘
template<unsigned int N>
struct Factorial {
    // 递归：N! = N * (N-1)!
    static constexpr unsigned long long value = N * Factorial<N - 1>::value;
};

// 递归终止条件：0! = 1 (模板特化)
template<>
struct Factorial<0> {
    static constexpr unsigned long long value = 1;
};

// 编译期斐波那契数列
template<unsigned int N>
struct Fibonacci {
    static constexpr unsigned long long value =
        Fibonacci<N - 1>::value + Fibonacci<N - 2>::value;
};

template<>
struct Fibonacci<0> {
    static constexpr unsigned long long value = 0;
};

template<>
struct Fibonacci<1> {
    static constexpr unsigned long long value = 1;
};

int main() {
    // 这些计算在编译期完成，运行时只是读取常量
    std::cout << "5!  = " << Factorial<5>::value << std::endl;     // 120
    std::cout << "10! = " << Factorial<10>::value << std::endl;    // 3628800
    std::cout << "20! = " << Factorial<20>::value << std::endl;    // 2432902008176640000

    std::cout << "Fib(10) = " << Fibonacci<10>::value << std::endl; // 55
    std::cout << "Fib(20) = " << Fibonacci<20>::value << std::endl; // 6765

    // 编译期计算可以直接用于数组大小、模板参数等场景
    int arr[Factorial<5>::value];  // 分配120个int的数组（编译期确定）

    return 0;
}

九、decltype 与 auto 返回类型推导

decltype 用于查询表达式的类型（不对表达式求值），auto 用于自动推导类型：

---------------------------------------------------------------
#include <iostream>
#include <vector>
#include <type_traits>

// decltype 用作返回类型后置语法
template<typename T1, typename T2>
auto add(T1 a, T2 b) -> decltype(a + b) {
    return a + b;
}

// C++14 起可以直接用 auto
template<typename T1, typename T2>
auto multiply(T1 a, T2 b) {
    return a * b;
}

// decltype 保留引用性质
int globalVar = 100;
decltype(globalVar) ref = globalVar;     // int —— 不是引用
decltype((globalVar)) ref2 = globalVar;   // int& —— 双括号得到引用

// 实际用法：模板中推导元素类型
template<typename Container>
void processContainer(const Container& c) {
    // 推导容器中元素的类型
    using ElementType = decltype(*c.begin());

    std::cout << "Element type size: " << sizeof(ElementType) << std::endl;

    // 如果需要 const 版本
    using ConstElementType = typename Container::value_type;
}

int main() {
    auto result1 = add(3, 4.5);      // decltype(3 + 4.5) => double
    auto result2 = multiply(2, 3.5); // double

    std::cout << result1 << " | " << result2 << std::endl;

    std::vector<int> vec = {1, 2, 3, 4, 5};
    processContainer(vec);

    return 0;
}

十、SFINAE 概念简介与 enable_if

SFINAE = Substitution Failure Is Not An Error（替换失败不是错误）
这是C++模板的一条核心原则：在模板参数替换过程中，如果某个候选函数
产生了不合法代码，编译器不会报错，而是静默地跳过该候选函数。

---------------------------------------------------------------
#include <iostream>
#include <type_traits>
#include <vector>

// enable_if 的基本用法：条件编译
// std::enable_if 在条件为 true 时暴露 type 成员，否则不暴露

// 仅对整数类型有效的函数
template<typename T>
typename std::enable_if<std::is_integral<T>::value, T>::type
safeDivide(T a, T b) {
    if(b == 0) throw std::runtime_error("Division by zero");
    return a / b;
}

// 仅对浮点类型有效的函数
template<typename T>
typename std::enable_if<std::is_floating_point<T>::value, T>::type
safeDivide(T a, T b) {
    if(std::abs(b) < 1e-10)
        throw std::runtime_error("Division by near-zero");
    return a / b;
}

// C++14 起有更简洁的写法
template<typename T>
std::enable_if_t<std::is_integral_v<T>, T>
multiplyByTwo(T x) {
    return x * 2;
}

// 另一个 SFINAE 示例：检测一个类型是否有 size() 方法
template<typename T>
auto hasSize(int) -> decltype(std::declval<T>().size(), std::true_type{});

template<typename T>
std::false_type hasSize(...);

template<typename T>
struct HasSizeMethod : decltype(hasSize<T>(0)) {};

int main() {
    // 整数版本
    std::cout << safeDivide(10, 3) << std::endl;      // 3
    // 浮点版本
    std::cout << safeDivide(10.0, 3.0) << std::endl;  // 3.33333

    std::cout << multiplyByTwo(21) << std::endl;       // 42

    std::cout << "vector has size(): "
              << HasSizeMethod<std::vector<int>>::value << std::endl; // 1
    std::cout << "int has size(): "
              << HasSizeMethod<int>::value << std::endl;              // 0

    return 0;
}

十一、Concepts (C++20) 简要介绍

Concepts 是 C++20 引入的模板约束机制，用于对模板参数施加编译期约束，
使模板错误信息更加清晰（替代 SFINAE 的笨重语法）：

---------------------------------------------------------------
// C++20 Concepts 示例（需要 C++20 编译器）
#if __cplusplus >= 202002L

#include <iostream>
#include <concepts>
#include <type_traits>

// 定义一个 Concept：类型必须支持 + 运算符并返回相同类型
template<typename T>
concept Addable = requires(T a, T b) {
    { a + b } -> std::same_as<T>;
};

// 定义一个 Concept：必须是数字类型
template<typename T>
concept Numeric = std::integral<T> || std::floating_point<T>;

// 使用 Concept 约束模板参数
template<Numeric T>
T square(T x) {
    return x * x;
}

// 定义必须支持 begin()/end() 的 Concept
template<typename T>
concept Iterable = requires(T t) {
    t.begin();
    t.end();
    typename T::value_type;
};

// 对支持迭代的类型打印
template<Iterable T>
void printElements(const T& container) {
    for(const auto& elem : container)
        std::cout << elem << " ";
    std::cout << std::endl;
}

int main() {
    std::cout << square(5) << std::endl;       // 25
    std::cout << square(2.5) << std::endl;     // 6.25

    std::vector<int> v = {1, 2, 3, 4, 5};
    printElements(v);                           // 1 2 3 4 5

    // printElements(42);  // 编译错误！int 不满足 Iterable concept
    return 0;
}

#endif

案例1: 通用比较器 —— 比赛评分系统排序
---------------------------------------------------------------
场景：体育比赛评委打分系统，需要比较不同选手的得分。得分可以是
int（田径计时）、double（体操评分）、string（比赛等级）。

知识点融入：
- Ch1 命名空间：namespace ScoreSystem {}
- Ch3 动态内存：动态分配选手成绩数组
- Ch7 运算符重载：>(已支持)

---------------------------------------------------------------
#include <iostream>
#include <string>
#include <cstring>
#include <vector>
#include <algorithm>

namespace ScoreSystem {

    // 通用比较模板：返回较大值
    template<typename T>
    T maxScore(T a, T b) {
        return (a > b) ? a : b;   // 依赖 operator>
    }

    // 泛型版本：返回较小值
    template<typename T>
    T minScore(T a, T b) {
        return (a < b) ? a : b;
    }

    // 特化版本：const char* 字符串比较
    // const char* 直接比较的是地址而非内容，必须特化处理
    template<>
    const char* maxScore<const char*>(const char* a, const char* b) {
        return (std::strcmp(a, b) > 0) ? a : b;
    }

    template<>
    const char* minScore<const char*>(const char* a, const char* b) {
        return (std::strcmp(a, b) < 0) ? a : b;
    }

    // 返回多个分数中的最大值（可变参数模板）
    template<typename T>
    T maxOfAll(T first) {
        return first;
    }

    template<typename T, typename... Args>
    T maxOfAll(T first, Args... rest) {
        T maxRest = maxOfAll(rest...);
        return maxScore(first, maxRest);
    }

    // 选手类（融入 Ch4 类与对象, Ch7 运算符重载）
    class Contestant {
    private:
        std::string name;
        double* scores;       // Ch3 动态内存
        int scoreCount;
        int id;
        static int nextId;    // 静态成员

    public:
        Contestant(const std::string& n, int count)
            : name(n), scoreCount(count), id(nextId++) {
            scores = new double[count];  // 动态分配
        }

        Contestant(const Contestant& other)   // 拷贝构造
            : name(other.name), scoreCount(other.scoreCount), id(nextId++) {
            scores = new double[scoreCount];
            for(int i = 0; i < scoreCount; ++i)
                scores[i] = other.scores[i];
        }

        Contestant& operator=(const Contestant& other) {
            if(this != &other) {
                name = other.name;
                scoreCount = other.scoreCount;
                delete[] scores;
                scores = new double[scoreCount];
                for(int i = 0; i < scoreCount; ++i)
                    scores[i] = other.scores[i];
            }
            return *this;
        }

        ~Contestant() { delete[] scores; }

        void setScore(int index, double value) {
            if(index >= 0 && index < scoreCount)
                scores[index] = value;
        }

        double getAverage() const {
            double total = 0;
            for(int i = 0; i < scoreCount; ++i)
                total += scores[i];
            return scoreCount > 0 ? total / scoreCount : 0;
        }

        // 运算符重载：按平均分比较 (Ch7)
        bool operator<(const Contestant& other) const {
            return getAverage() < other.getAverage();
        }

        bool operator>(const Contestant& other) const {
            return getAverage() > other.getAverage();
        }

        friend std::ostream& operator<<(std::ostream& os,
                                         const Contestant& c) {
            os << "选手 #" << c.id << " [" << c.name
               << "] 平均分: " << c.getAverage();
            return os;
        }

        const std::string& getName() const { return name; }
        int getId() const { return id; }
    };

    int Contestant::nextId = 1000;

} // namespace ScoreSystem

int main() {
    using namespace ScoreSystem;

    // 测试通用比较器
    std::cout << "===== 比赛评分系统 =====" << std::endl;

    // int 类型比较（田径百米成绩）
    std::cout << "100米跑最快: "
              << maxScore(1030, 987) << " (百分秒)" << std::endl;

    // double 类型比较（体操评分）
    std::cout << "体操最高分: "
              << maxScore(9.85, 9.92) << std::endl;

    // 字符串比较（比赛等级）
    const char* grade1 = "金牌";
    const char* grade2 = "银牌";
    std::cout << "奖牌等级: "
              << maxScore(grade1, grade2) << std::endl;

    // 可变参数比较
    std::cout << "三项综合最高: "
              << maxOfAll(9.1, 9.5, 9.3, 8.9, 9.6) << std::endl;

    // 选手管理
    Contestant c1("张三", 5);
    Contestant c2("李四", 5);

    double scores1[] = {9.2, 9.5, 9.1, 9.8, 9.3};
    double scores2[] = {9.0, 9.4, 9.6, 9.2, 9.5};

    for(int i = 0; i < 5; ++i) {
        c1.setScore(i, scores1[i]);
        c2.setScore(i, scores2[i]);
    }

    std::cout << c1 << std::endl;
    std::cout << c2 << std::endl;

    // 用模板比较器找出更好的选手
    Contestant winner = maxScore(c1, c2);
    std::cout << "胜出者: " << winner.getName() << std::endl;

    return 0;
}

案例2: 通用容器打印器 —— 数据库查询结果展示
---------------------------------------------------------------
场景：开发一个数据库管理工具，需要把不同类型的查询结果
（整数ID列表、字符串名称列表、浮点数价格列表等）格式化显示。

知识点融入：
- Ch1 命名空间：namespace DBUtils {}
- Ch2 预处理器：DEBUG 宏控制调试输出
- Ch3 动态内存：迭代器遍历
- 模板 + decltype/auto

---------------------------------------------------------------
#include <iostream>
#include <vector>
#include <list>
#include <set>
#include <map>
#include <string>
#include <iomanip>

// 第二章：预处理器宏用于调试级别
#define DB_DEBUG_LEVEL 1

#if DB_DEBUG_LEVEL > 0
    #define DB_LOG(msg) std::cout << "[DBG] " << msg << std::endl
#else
    #define DB_LOG(msg)
#endif

namespace DBUtils {

    // 通用容器打印模板函数
    template<typename Container>
    void printContainer(const Container& c, const std::string& title = "") {
        if(!title.empty())
            std::cout << "--- " << title << " ---" << std::endl;

        std::cout << "[";
        size_t count = 0;
        for(auto it = c.begin(); it != c.end(); ++it) {
            if(count++ > 0) std::cout << ", ";
            std::cout << *it;
        }
        std::cout << "]" << std::endl;
    }

    // 特化：map 容器单独处理(key-value对)
    template<typename K, typename V>
    void printContainer(const std::map<K, V>& m,
                         const std::string& title = "") {
        if(!title.empty())
            std::cout << "--- " << title << " ---" << std::endl;

        for(const auto& pair : m) {
            std::cout << "  " << pair.first << " => "
                      << pair.second << std::endl;
        }
    }

    // 用 decltype 推导元素类型并格式化
    template<typename Container>
    void printWithTypeInfo(const Container& c) {
        using ElementType = decltype(*c.begin());
        std::cout << "容器元素类型大小: " << sizeof(ElementType)
                  << " bytes, 元素数量: " << c.size() << std::endl;
        printContainer(c);
    }

    // 表格形式打印（用于数据库查询结果）
    template<typename Container>
    void printAsTable(const Container& c,
                      int colWidth = 12,
                      int columnsPerRow = 4) {
        int count = 0;
        for(const auto& elem : c) {
            std::cout << std::setw(colWidth) << std::left << elem;
            if(++count % columnsPerRow == 0)
                std::cout << std::endl;
        }
        if(count % columnsPerRow != 0)
            std::cout << std::endl;
    }

} // namespace DBUtils

int main() {
    using namespace DBUtils;

    DB_LOG("开始数据库查询演示");

    // 模拟不同类型查询结果
    std::vector<int> userIds = {1001, 1002, 1003, 1004, 1005};
    std::list<std::string> cityNames =
        {"北京", "上海", "广州", "深圳", "杭州", "成都"};
    std::set<double> productPrices = {19.99, 29.50, 99.00, 149.99, 9.90};

    std::map<int, std::string> userMap = {
        {1001, "张三"}, {1002, "李四"}, {1003, "王五"}
    };

    // 打印查询结果
    printContainer(userIds, "用户ID查询结果");
    printContainer(cityNames, "城市查询结果");
    printContainer(productPrices, "商品价格查询结果(已排序)");
    printContainer(userMap, "用户映射表");

    // 带有类型信息的打印
    printWithTypeInfo(userIds);

    // 表格格式打印
    std::cout << "\n=== 表格格式输出 ===" << std::endl;
    std::vector<std::string> products = {
        "手机", "电脑", "平板", "耳机", "手表", "音箱",
        "鼠标", "键盘", "显示器", "充电器", "数据线", "支架"
    };
    printAsTable(products, 15, 4);

    DB_LOG("查询演示结束");

    return 0;
}

案例3: 可变参数日志系统 —— 服务器多级别日志输出
---------------------------------------------------------------
场景：后端服务器需要多级别日志系统，支持任意数量的格式化参数，
按级别过滤输出。类似企业级日志框架。

知识点融入：
- Ch1 命名空间：namespace Logger {}
- Ch2 预处理器：LOG_LEVEL 宏控制编译期日志级别
- 本章：可变参数模板 + 折叠表达式

---------------------------------------------------------------
#include <iostream>
#include <string>
#include <sstream>
#include <ctime>
#include <chrono>
#include <iomanip>

// 第二章：用预处理器控制日志级别
#ifndef LOG_LEVEL
    #define LOG_LEVEL 0   // 0=ALL, 1=DEBUG, 2=INFO, 3=WARN, 4=ERROR, 5=FATAL
#endif

namespace Logger {

    // 日志级别枚举
    enum class Level {
        ALL = 0, DEBUG = 1, INFO = 2,
        WARN = 3, ERROR = 4, FATAL = 5
    };

    // 获取级别字符串
    const char* levelToString(Level lvl) {
        switch(lvl) {
            case Level::DEBUG: return "DEBUG";
            case Level::INFO:  return "INFO ";
            case Level::WARN:  return "WARN ";
            case Level::ERROR: return "ERROR";
            case Level::FATAL: return "FATAL";
            default:           return "ALL  ";
        }
    }

    // 获取当前时间字符串
    std::string currentTime() {
        auto now = std::chrono::system_clock::now();
        auto timeT = std::chrono::system_clock::to_time_t(now);
        auto ms = std::chrono::duration_cast<std::chrono::milliseconds>(
            now.time_since_epoch()) % 1000;

        std::stringstream ss;
        ss << std::put_time(std::localtime(&timeT), "%Y-%m-%d %H:%M:%S");
        ss << '.' << std::setfill('0') << std::setw(3) << ms.count();
        return ss.str();
    }

    // 将任意参数转为字符串的核心函数
    template<typename T>
    std::string toString(const T& value) {
        std::stringstream ss;
        ss << value;
        return ss.str();
    }

    // 日志记录：单个参数
    template<typename T>
    void log(Level lvl, const T& msg) {
        if(static_cast<int>(lvl) < LOG_LEVEL) return;  // 级别过滤
        std::cout << "[" << currentTime() << "]"
                  << "[" << levelToString(lvl) << "] "
                  << toString(msg) << std::endl;
    }

    // 日志记录：可变参数 —— 递归展开
    template<typename T, typename... Args>
    void log(Level lvl, const T& first, const Args&... rest) {
        if(static_cast<int>(lvl) < LOG_LEVEL) return;

        std::stringstream ss;
        ss << "[" << currentTime() << "]"
           << "[" << levelToString(lvl) << "] ";

        // 用折叠表达式拼接所有参数
        ss << toString(first);
        ((ss << " " << toString(rest)), ...);

        std::cout << ss.str() << std::endl;
    }

    // 批量日志：一次性输出多条同级别消息（用 initializer_list）
    template<typename... Args>
    void batchLog(Level lvl, Args... msgs) {
        if(static_cast<int>(lvl) < LOG_LEVEL) return;
        std::cout << "[" << currentTime() << "]"
                  << "[" << levelToString(lvl) << "] === 批量消息 ==="
                  << std::endl;
        int count = 0;
        // 折叠表达式依次输出
        ((std::cout << "  [" << ++count << "] "
                    << toString(msgs) << std::endl), ...);
    }

    // 统计日志中使用折叠表达式求和（日志规模统计）
    template<typename... Args>
    size_t totalMessageLength(Args... msgs) {
        // 所有消息字符串长度的和

案例4: 通用数学库 —— 3D游戏物理引擎
---------------------------------------------------------------
场景：3D游戏开发中需要向量运算（位置、速度、加速度等），
不同维度(N)和不同类型(float/double)的向量。

知识点融入：
- Ch1 命名空间：namespace MathLib {}
- Ch3 动态内存：存储向量分量
- Ch6 多态：可分派操作
- Ch7 运算符重载：+、-、*、/、[]等
- 本章：模板类 / enable_if 约束 / 特化

---------------------------------------------------------------
#include <iostream>
#include <cmath>
#include <type_traits>
#include <stdexcept>
#include <initializer_list>

namespace MathLib {

    // 通用向量模板类
    // enable_if 约束：只允许算术类型（整数和浮点）
    template<typename T, int N,
             typename = std::enable_if_t<std::is_arithmetic<T>::value>>
    class Vector {
    private:
        T* components;   // Ch3 动态内存分配

    public:
        // 构造函数
        Vector() {
            components = new T[N]{};
        }

        Vector(std::initializer_list<T> list) {
            if(list.size() != static_cast<size_t>(N))
                throw std::invalid_argument("维度不匹配");
            components = new T[N];
            int i = 0;
            for(auto& val : list) components[i++] = val;
        }

        // 拷贝构造
        Vector(const Vector& other) {
            components = new T[N];
            for(int i = 0; i < N; ++i)
                components[i] = other.components[i];
        }

        // 赋值运算符
        Vector& operator=(const Vector& other) {
            if(this != &other) {
                for(int i = 0; i < N; ++i)
                    components[i] = other.components[i];
            }
            return *this;
        }

        // 析构函数
        ~Vector() { delete[] components; }

        // 获取维度
        static constexpr int dimension() { return N; }

        // 下标访问运算符重载 (Ch7)
        T& operator[](int index) {
            if(index < 0 || index >= N)
                throw std::out_of_range("索引越界");
            return components[index];
        }

        const T& operator[](int index) const {
            if(index < 0 || index >= N)
                throw std::out_of_range("索引越界");
            return components[index];
        }

        // 向量加法
        Vector operator+(const Vector& other) const {
            Vector result;
            for(int i = 0; i < N; ++i)
                result.components[i] = components[i] + other.components[i];
            return result;
        }

        // 向量减法
        Vector operator-(const Vector& other) const {
            Vector result;
            for(int i = 0; i < N; ++i)
                result.components[i] = components[i] - other.components[i];
            return result;
        }

        // 标量乘法
        Vector operator*(T scalar) const {
            Vector result;
            for(int i = 0; i < N; ++i)
                result.components[i] = components[i] * scalar;
            return result;
        }

        // 标量除法
        Vector operator/(T scalar) const {
            if(scalar == 0) throw std::runtime_error("除数为零");
            Vector result;
            for(int i = 0; i < N; ++i)
                result.components[i] = components[i] / scalar;
            return result;
        }

        // 点积 (Dot Product)
        T dot(const Vector& other) const {
            T result = 0;
            for(int i = 0; i < N; ++i)
                result += components[i] * other.components[i];
            return result;
        }

        // 向量长度（模）
        double magnitude() const {
            double sum = 0;
            for(int i = 0; i < N; ++i)
                sum += components[i] * components[i];
            return std::sqrt(sum);
        }

        // 归一化
        Vector normalized() const {
            double mag = magnitude();
            if(mag < 1e-10) return *this;
            Vector result;
            for(int i = 0; i < N; ++i)
                result.components[i] =
                    static_cast<T>(components[i] / mag);
            return result;
        }

        // 流输出
        friend std::ostream& operator<<(std::ostream& os,
                                         const Vector& v) {
            os << "Vector" << N << "(";
            for(int i = 0; i < N; ++i) {
                if(i > 0) os << ", ";
                os << v.components[i];
            }
            os << ")";
            return os;
        }
    };

    // 模板特化：Vector<bool, N> 做位压缩存储
    // 每个分量只有 true/false，用一个字节可存8个维度
    template<int N>
    class Vector<bool, N> {
    private:
        static constexpr int bytes_needed = (N + 7) / 8;
        unsigned char data[bytes_needed] = {};

        bool getBit(int index) const {
            int byteIdx = index / 8;
            int bitIdx = index % 8;
            return (data[byteIdx] >> bitIdx) & 1;
        }

        void setBit(int index, bool value) {
            int byteIdx = index / 8;
            int bitIdx = index % 8;
            if(value)
                data[byteIdx] |= (1 << bitIdx);    // 置位
            else
                data[byteIdx] &= ~(1 << bitIdx);   // 清零
        }

    public:
        static constexpr int dimension() { return N; }

        // 代理类用于下标访问
        class BitRef {
        private:
            unsigned char* bytePtr;
            int bitIdx;
        public:
            BitRef(unsigned char* ptr, int idx)
                : bytePtr(ptr), bitIdx(idx) {}

            operator bool() const {
                return (*bytePtr >> bitIdx) & 1;
            }

            BitRef& operator=(bool val) {
                if(val)
                    *bytePtr |= (1 << bitIdx);
                else
                    *bytePtr &= ~(1 << bitIdx);
                return *this;
            }
        };

        BitRef operator[](int index) {
            return BitRef(&data[index / 8], index % 8);
        }

        bool operator[](int index) const {
            return getBit(index);
        }

        friend std::ostream& operator<<(std::ostream& os,
                                         const Vector& v) {
            os << "BoolVector" << N << "(";
            for(int i = 0; i < N; ++i) {
                if(i > 0) os << ", ";
                os << (v.getBit(i) ? "1" : "0");
            }
            os << ")";
            return os;
        }
    };

    // 跨积（叉积）—— 仅3D有意义
    template<typename T>
    Vector<T, 3> cross(const Vector<T, 3>& a, const Vector<T, 3>& b) {
        return Vector<T, 3>({
            a[1] * b[2] - a[2] * b[1],
            a[2] * b[0] - a[0] * b[2],
            a[0] * b[1] - a[1] * b[0]
        });
    }

} // namespace MathLib

int main() {
    using namespace MathLib;

    std::cout << "===== 3D游戏物理引擎向量演示 =====" << std::endl;

    // 3D 位置向量
    Vector<double, 3> position({0.0, 0.0, 0.0});
    Vector<double, 3> velocity({10.0, 5.0, 2.5});
    Vector<double, 3> gravity({0.0, -9.8, 0.0});
    Vector<double, 3> force({3.0, 0.0, 1.0});

    std::cout << "初始位置: " << position << std::endl;
    std::cout << "速度: " << velocity << std::endl;
    std::cout << "重力: " << gravity << std::endl;

    // 物理计算：新速度 = 旧速度 + 加速度*时间(简化)
    Vector<double, 3> newVelocity = velocity + gravity * 0.016; // 16ms帧
    Vector<double, 3> newPosition = position + newVelocity * 0.016;

    std::cout << "新速度: " << newVelocity << std::endl;
    std::cout << "新位置: " << newPosition << std::endl;

    // 点积：计算力在速度方向的分量（做功计算）
    double work = force.dot(velocity);
    std::cout << "力在速度方向的功: " << work << std::endl;

    // 归一化方向向量
    auto direction = velocity.normalized();
    std::cout << "速度方向单位向量: " << direction << std::endl;

    // 叉积：计算垂直于两个向量的向量（如法向量）
    Vector<double, 3> up({0.0, 1.0, 0.0});
    auto right = cross(up, direction);
    std::cout << "右侧向量: " << right << std::endl;

    // 4D 颜色向量（RGBA）
    Vector<float, 4> color({0.2f, 0.5f, 0.8f, 1.0f});
    std::cout << "颜色(RGBA): " << color << std::endl;

    // 特化：bool 向量（碰撞检测结果）
    Vector<bool, 8> collisionFlags;
    collisionFlags[0] = true;   // 与地面碰撞
    collisionFlags[1] = false;  // 未与墙壁碰撞
    collisionFlags[3] = true;   // 与敌人碰撞
    std::cout << "碰撞标记: " << collisionFlags << std::endl;

    return 0;
}

案例5: 通用工厂模式 —— 插件式应用框架

场景：开发一个插件式应用框架，需要根据配置文件动态创建不同类型的
插件对象（如滤镜插件、输出插件、输入插件），使用工厂模式统一管理。

#include <iostream>
#include <memory>
#include <string>
#include <map>
#include <functional>
#include <type_traits>
#include <vector>
#include <sstream>

namespace PluginFramework {

    // ===== 插件基类体系 (Ch4, Ch5, Ch6) =====

    class Plugin {
    protected:
        std::string name;
        std::string version;
    public:
        Plugin(const std::string& n, const std::string& v)
            : name(n), version(v) {}
        virtual ~Plugin() = default;

        virtual void initialize() {
            std::cout << "[Plugin] " << name << " v" << version
                      << " 初始化" << std::endl;
        }

        virtual void execute() = 0;  // 纯虚函数

        virtual std::string typeName() const = 0;

        const std::string& getName() const { return name; }
        const std::string& getVersion() const { return version; }
    };

    // 滤镜插件
    class FilterPlugin : public Plugin {
    private:
        std::string filterType;
    public:
        FilterPlugin(const std::string& n, const std::string& v,
                     const std::string& ftype)
            : Plugin(n, v), filterType(ftype) {}

        void execute() override {
            std::cout << "[Filter] 应用滤镜: " << filterType << std::endl;
        }

        std::string typeName() const override { return "Filter"; }
    };

    // 输出插件
    class OutputPlugin : public Plugin {
    private:
        std::string format;
    public:
        OutputPlugin(const std::string& n, const std::string& v,
                     const std::string& fmt)
            : Plugin(n, v), format(fmt) {}

        void execute() override {
            std::cout << "[Output] 输出格式: " << format << std::endl;
        }

        std::string typeName() const override { return "Output"; }
    };

    // 输入插件
    class InputPlugin : public Plugin {
    private:
        std::string sourceType;
    public:
        InputPlugin(const std::string& n, const std::string& v,
                    const std::string& src)
            : Plugin(n, v), sourceType(src) {}

        void execute() override {
            std::cout << "[Input] 数据来源: " << sourceType << std::endl;
        }

        std::string typeName() const override { return "Input"; }
    };

    // ===== 模板元编程：编译期类型检查 (本章核心) =====

    // 检查 T 是否继承自 Plugin
    template<typename T>
    struct IsPlugin {
    private:
        // SFINAE: 如果 T 可以转换为 Plugin*，选择这个重载
        template<typename U>
        static auto test(int) -> decltype(
            static_cast<Plugin*>(std::declval<U*>()), std::true_type{}
        );
        // 否则选择这个
        template<typename U>
        static std::false_type test(...);
    public:
        static constexpr bool value = decltype(test<T>(0))::value;
    };

    // C++17 版本可以简化为：
    // template<typename T>
    // constexpr bool IsPlugin_v = std::is_base_of_v<Plugin, T>;

    // ===== 通用工厂函数 (可变参数模板 + 完美转发) =====

    template<typename T, typename... Args,
             typename = std::enable_if_t<IsPlugin<T>::value>>
    std::unique_ptr<T> create(Args&&... args) {
        // 使用 std::forward 完美转发构造参数
        auto plugin = std::make_unique<T>(std::forward<Args>(args)...);

        std::cout << "[Factory] 创建插件: " << plugin->getName()
                  << " (类型: " << plugin->typeName() << ")"
                  << std::endl;
        return plugin;
    }

    // 重载：带初始化调用的创建
    template<typename T, typename... Args,
             typename = std::enable_if_t<IsPlugin<T>::value>>
    std::unique_ptr<T> createAndInit(Args&&... args) {
        auto plugin = create<T>(std::forward<Args>(args)...);
        plugin->initialize();
        return plugin;
    }

    // ===== 插件管理器 (融入 Ch1 命名空间, Ch3 智能指针) =====

    class PluginManager {
    private:
        std::map<std::string, std::unique_ptr<Plugin>> plugins;
        static int pluginCount;

    public:
        template<typename T, typename... Args>
        T* registerPlugin(Args&&... args) {
            auto plugin = createAndInit<T>(std::forward<Args>(args)...);
            std::string key = plugin->getName();
            T* rawPtr = plugin.get();
            plugins[key] = std::move(plugin);
            pluginCount++;
            return rawPtr;
        }

        void executeAll() {
            std::cout << "\n=== 执行所有插件 (共"
                      << plugins.size() << "个) ===" << std::endl;
            for(auto& pair : plugins) {
                pair.second->execute();
            }
        }

        Plugin* findPlugin(const std::string& name) {
            auto it = plugins.find(name);
            if(it != plugins.end())
                return it->second.get();
            return nullptr;
        }

        static int getCount() { return pluginCount; }
    };

    int PluginManager::pluginCount = 0;

} // namespace PluginFramework

int main() {
    using namespace PluginFramework;

    std::cout << "===== 插件式应用框架演示 =====" << std::endl;

    PluginManager manager;

    // 创建并注册不同类型的插件
    auto* filter = manager.registerPlugin<FilterPlugin>(
        "锐化滤镜", "1.0.0", "UnsharpMask"
    );
    auto* output = manager.registerPlugin<OutputPlugin>(
        "PNG输出", "2.1.0", "PNG"
    );
    auto* input = manager.registerPlugin<InputPlugin>(
        "摄像头输入", "1.0.0", "USB Camera"
    );

    // 独立创建一个插件（不注册到管理器）
    auto standalone = create<FilterPlugin>("测试滤镜", "0.0.1", "Test");
    standalone->initialize();
    standalone->execute();

    // 执行所有注册的插件（多态调用）
    manager.executeAll();

    // 查找并执行特定插件
    Plugin* found = manager.findPlugin("PNG输出");
    if(found) {
        std::cout << "\n找到插件: " << found->getName() << std::endl;
        found->execute();
    }

    // 模板元编程验证：编译期类型检查
    static_assert(IsPlugin<FilterPlugin>::value,
                  "FilterPlugin 必须是 Plugin 的派生类");
    static_assert(IsPlugin<OutputPlugin>::value,
                  "OutputPlugin 必须是 Plugin 的派生类");

    // 以下代码会导致编译错误（static_assert 阻止）：
    // static_assert(IsPlugin<std::string>::value,
    //               "std::string 不是 Plugin 的派生类");

    std::cout << "\n插件总数: " << PluginManager::getCount() << std::endl;

    // 演示完美转发：传递右值
    std::string tempName = "临时插件";
